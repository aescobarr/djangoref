New in version {{ version }}

  • Hyerarchy search on site names: the option 'inside' of the filter allows the user to filter all site names contained in another site name. For instance, filtering for site names 'inside' UK will return all site names with parent site 'UK'
  • Centroid visualization in site names: on adding geometry to a site name, the centroid is indicated by a marker
  • New centroid calculation algorithm: if the centroid of a site name falls outside of its associated geometry, the centroid is corrected and moved to the nearest point on the geometry
  • New total uncertainty and metadata fields: the following uncertainty and metadata fields have been added to site names: Maximum depth/height(m), Minimum depth/height(m), Georeferencing calculator metadata string, Georeferencing calculator uncertainty (m)
  • Excel file support for site names import: the import tool now supports this file format
  • New visualizations for 'state' site names: count of number of site names by state and tree view of states have been added to Tools -> Usage charts
  • Utility menu edition interface: the admin user can now edit (add, delete, edit) links on the utility menu
Bug fixes/improvements:
  • Fixed bug which prevented refresh of the site layer when filtering by author
  • Fixed bug which prevented some recent sites showing (join with countries table)
  • Disabled automatic load of wms servers on resources form
